NeuroOS: Kernel Engineering & Biomimetic Systems

GitHub Repository

An experimental operating system kernel exploring biological memory models. Implemented Hebbian Allocation (where malloc acts as a synapse), Synaptic Pruning garbage collection, and Neural Reinforcement routines, treating low-level memory as a plastic, synaptic connection. This approach blurs the boundary between traditional OS design and neuroscience, yielding insights into both domains.

Lévy Flights: Statistical Mechanics

GitHub Repository

"On the Absence of Intrinsic Ergodicity in Heavy-Tailed Sums." Proved that extreme fluctuations in Cauchy distributions dampen into predictable, stationary measures when mapped into a logarithmic coordinate system. Backed by a custom C-based computer-assisted proof. Challenges classical assumptions about the long-term behavior of heavy-tailed stochastic processes. (Nonlinear Sciences Submission).
